== Description ==
Through this plugin, user can able to combine the repeated tags under the single group tag name

* Users can able to add repeated tags under a group name
* Users can able to add up to Ten custom group name and tags
* Users can select the group name in Add New Post section
* After publishing the post, all tags under the respective Group name will be added to the tag section.
* User can also edit the multiple tag in quick edit
* User can also do bulk edit (new)

For More information : http://buffercode.com/multiple-tags-wordpress-plugin/

== Installation ==
1. Upload the "multiple-tags" directory to the plugins directory.
2. Go to the plugins setting page and activate "Multiple Tags". 
3. Go to Settings --> Multiple Tag --> Fill the Group name and Tags [ Tags should be Comma Separation  ]. Find the sample input in setting page.
4. Users can able to view Multiple Tag in Posting area.

For More information : http://buffercode.com/multiple-tags-wordpress-plugin/
